Herramientas Azure para Gestión de Datos

Clasificación de servicios según su rol en la Arquitectura Medallion: desde la ingesta de datos crudos hasta el consumo analítico.

BRONCE — Datos Crudos → PLATA — Datos Limpios → ORO — Datos Listos
BRONCE — Ingesta y Almacenamiento Crudo
Captura y almacena datos tal como llegan, sin transformación
Azure Data Factory
Orquestador ETL

Extrae datos automáticamente desde múltiples fuentes (bases de datos, APIs, archivos) y los mueve hacia el Data Lake. Es el punto de entrada principal de datos.

Ver detalles
Casos de uso
Migración de datos on-premise a Azure, sincronización entre bases de datos, carga incremental diaria a Data Lake, orquestación de pipelines complejos con dependencias.
Se conecta con
Data Lake Gen2 Databricks Synapse SQL Database
Azure Event Hubs / IoT Hub
Ingesta en Tiempo Real

Captura datos de alta frecuencia en tiempo real: sensores, telemetría, streams. Ideal para datos que llegan continuamente como oleaje, viento o movimiento de naves.

Ver detalles
Casos de uso
Telemetría IoT de sensores industriales, ingestión de logs de aplicaciones, streaming de eventos de clickstream, procesamiento de datos GPS en tiempo real.
Se conecta con
Stream Analytics Data Lake Gen2 Functions
Azure Data Lake Storage Gen2
Repositorio Principal

Almacén centralizado de todos los datos: crudos (Bronce) y semi-procesados (Plata). Gran escala, bajo costo, compatible con Spark y Hadoop.

Ver detalles
Casos de uso
Almacenamiento centralizado de datos crudos y procesados, data lake corporativo para analytics, almacén de archivos Parquet/Delta para Spark, backup de datos transaccionales.
Se conecta con
Data Factory Databricks Synapse Purview
PLATA — Procesamiento y Transformación
Limpia, transforma y estandariza los datos crudos
Azure Databricks
Motor de Análisis y ETL

Plataforma avanzada de análisis basada en Apache Spark. Limpia datos, aplica reglas de calidad, hace transformaciones complejas y prepara datasets para modelos de Machine Learning.

Ver detalles
Casos de uso
Limpieza y normalización de datos a escala, feature engineering para ML, procesamiento batch/streaming con Delta Lake, análisis exploratorio colaborativo en notebooks.
Se conecta con
Data Lake Gen2 Data Factory Synapse Machine Learning
Azure Stream Analytics
Procesamiento Streaming

Procesa datos en tiempo real con latencia de milisegundos. Detecta patrones al vuelo, dispara alertas automáticas y transforma streams de sensores antes de almacenarlos.

Ver detalles
Casos de uso
Detección de anomalías en tiempo real, dashboards operacionales live, alertas automáticas por umbrales, transformación de streams IoT antes de persistirlos.
Se conecta con
Event Hubs Data Lake Gen2 Power BI Functions
ORO — Modelado y Consumo
Datos listos para análisis, reportes y modelos predictivos
Azure Synapse Analytics
Data Warehouse + Big Data

Plataforma unificada que combina Data Warehouse (SQL) con Big Data (Spark). Aquí se construyen los modelos analíticos y Data Marts listos para consumo por Power BI o Machine Learning.

Ver detalles
Casos de uso
Data warehouse empresarial, creación de Data Marts por área de negocio, consultas SQL a escala de petabytes, integración directa con Power BI para reportes en tiempo real.
Se conecta con
Data Lake Gen2 Databricks Power BI Purview
Azure Machine Learning
Modelos Predictivos e IA

Gestiona el ciclo completo de ML: entrenamiento, versionado y despliegue de modelos. Expone los modelos como endpoints (APIs) para que otras aplicaciones los consuman.

Ver detalles
Casos de uso
Modelos de predicción de demanda, detección de fraude, clasificación de documentos, análisis de sentimiento, MLOps con versionado y despliegue automatizado.
Se conecta con
Databricks Container Apps API Management Synapse
Power BI
Visualización y Reportes

Herramienta de Business Intelligence que consume datos de la capa Oro para generar dashboards, reportes automáticos e indicadores operacionales para las áreas del negocio.

Ver detalles
Casos de uso
Dashboards ejecutivos, reportes operacionales automatizados, KPIs en tiempo real, análisis self-service para usuarios de negocio, informes embebidos en aplicaciones.
Se conecta con
Synapse Databricks Stream Analytics Entra ID
TRANSVERSAL — Gobernanza, Seguridad y Automatización
Servicios que operan en todas las capas simultáneamente
Microsoft Purview
Gobernanza de Datos

Catálogo central de datos: qué existe, dónde está, quién lo usa. Gestiona el linaje (trazabilidad de origen a destino), calidad y clasificación por sensibilidad.

Ver detalles
Casos de uso
Catálogo de activos de datos empresarial, trazabilidad end-to-end de pipelines, clasificación automática de datos sensibles (PII, financieros), cumplimiento normativo.
Se conecta con
Data Lake Gen2 Synapse Data Factory Power BI
Azure Key Vault
Seguridad

Almacena de forma segura credenciales, contraseñas, claves de cifrado y certificados. Ningún servicio guarda sus propias contraseñas: todas vienen de aquí.

Ver detalles
Casos de uso
Rotación automática de secretos, almacenamiento de connection strings, certificados TLS para APIs, claves de cifrado para datos en reposo y tránsito.
Se conecta con
Data Factory Functions API Management Container Apps
Azure Monitor
Monitoreo

Observabilidad de toda la plataforma: métricas, logs y alertas de rendimiento. Supervisa pipelines, APIs y recursos de cómputo en tiempo real.

Ver detalles
Casos de uso
Alertas por fallos en pipelines ETL, monitoreo de latencia de APIs, dashboards de salud de infraestructura, diagnóstico de cuellos de botella en Spark jobs.
Se conecta con
Todos los servicios Logic Apps Functions
Azure Logic Apps
Automatización Low-Code

Crea flujos de trabajo automáticos sin código: enviar alertas por Teams/email, disparar acciones cuando un modelo detecta una anomalía, notificar a operaciones.

Ver detalles
Casos de uso
Notificaciones automáticas por Teams al fallar un pipeline, sincronización de datos entre SaaS, aprobaciones automatizadas, integración con SharePoint y Outlook.
Se conecta con
Monitor Data Factory Teams Outlook
Azure Functions
Cómputo Serverless

Ejecuta pequeñas piezas de código bajo demanda sin gestionar servidores. Ideal para auditorías, notificaciones automáticas y micro-tareas disparadas por eventos.

Ver detalles
Casos de uso
Procesamiento de archivos al subirse al blob, webhooks para integraciones externas, tareas programadas de limpieza, validación de datos en tiempo real.

Azure API Management
Gestión de APIs

Puerta de entrada controlada para exponer modelos y datos a aplicaciones internas o externas. Gestiona seguridad, límites de uso (throttling) y documentación de APIs.

Ver detalles
Casos de uso
Exposición segura de modelos ML como APIs, rate limiting para consumidores externos, portal de documentación para desarrolladores, monetización de datos como servicio.

Container Apps Functions Entra ID Key Vault
Azure Active Directory (Entra ID)
