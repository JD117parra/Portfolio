Responsabilidad compartida
Gestionado por Microsoft
IaaS
Infrastructure as a Service
Tú gestionas desde el sistema operativo hacia arriba: apps, datos, runtime, middleware y SO. Microsoft gestiona virtualización, servidores, almacenamiento y redes.
Ejemplo: Virtual Machines, VMSS, Azure Disk Storage
PaaS
Platform as a Service
Solo gestionas aplicaciones y datos. Microsoft gestiona todo lo demás: runtime, middleware, SO, virtualización, servidores y redes.
Ejemplo: App Service, Azure SQL, Azure Functions
SaaS
Software as a Service
Microsoft gestiona toda la infraestructura y la aplicación. Tú gestionas la configuración, los datos del usuario y el control de acceso.
Ejemplo: Microsoft 365, Dynamics 365, Power BI
